# Backend Scaffold

This folder is intentionally scaffold-only.

Planned contents:

- `src/` for API server code
- `src/config/` for environment and runtime config
- `src/routes/` for route modules
- `src/services/` for business logic
- `src/db/` for PostgreSQL connection and queries

Supabase will be used through a direct PostgreSQL connection string (`SUPABASE_DB_URL`) when implementation begins.
