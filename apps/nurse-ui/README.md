# Nurse UI Scaffold

This folder is intentionally scaffold-only.

Planned contents:

- `src/` for UI source code
- `src/components/` for shared UI components
- `src/screens/` for screen/page-level modules
- `src/services/` for API client calls to backend
- `assets/` for images/icons/fonts

This client is intended to become a React Native + Expo app (Expo Go compatible).
